# homework_Akhmetshin
Проект демонстрирует реалистичное управление 3D-шаром с физикой:
Плавное катание по сцене
Прыжки во время движения
Полноценное физическое взаимодействие

Основные возможности
Физически корректное движение
Динамические прыжки

Управление
Клавиша	Действие
W	Вперед
A	Влево
S	Назад
D	Вправо
